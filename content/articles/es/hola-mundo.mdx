# Bienvenido a MDX

Este es un artículo de ejemplo escrito en formato **MDX**. MDX te permite escribir JSX directamente en tus documentos markdown, haciéndolo increíblemente poderoso para crear documentación interactiva y publicaciones de blog.

## ¿Qué es MDX?

MDX es un formato que combina **Markdown** con **JSX**. Esto significa que puedes:

- Escribir contenido en Markdown
- Importar y usar componentes de React
- Crear ejemplos interactivos
- Construir contenido rico y atractivo

## Ejemplos de Código

Aquí hay un ejemplo simple de TypeScript:

```typescript
interface Usuario {
  nombre: string;
  email: string;
}

function saludarUsuario(usuario: Usuario): string {
  return `¡Hola, ${usuario.nombre}!`;
}

const usuario: Usuario = {
  nombre: "Enrique",
  email: "hola@ejemplo.com"
};

console.log(saludarUsuario(usuario));
```
