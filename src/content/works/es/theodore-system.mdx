# Theodore System

Theodore es un sistema impulsado por IA que transforma el proceso tradicional de escritura de tesis en un pipeline eficiente y asistido por IA. Simplifica todo el flujo de trabajo, desde los borradores iniciales en Markdown hasta los documentos finales con formato profesional en Adobe InDesign.

## Resumen del Proyecto

Creado bajo el principio de que las herramientas modernas de IA funcionan mejor con contenido modular y bien estructurado, Theodore ofrece un flujo de trabajo integral para autores. Al organizar una tesis en secciones y archivos manejables, los escritores pueden aprovechar la IA para la generación y revisión de contenido, mantener un control de versiones robusto y automatizar el proceso de publicación. Este sistema es ideal para estudiantes de doctorado, investigadores y escritores académicos que desean centrarse en la creación de contenido en lugar del formato manual.

## Características Clave

- **Escritura Impulsada por IA**: Un enfoque modular permite que las herramientas de IA procesen y mejoren el contenido de manera efectiva, desde la generación de borradores hasta el seguimiento de múltiples rondas de revisiones.
- **Gestión de Contenido Flexible**: Organiza tu tesis con carpetas de sección personalizables y archivos Markdown numerados para un orden automático, manteniendo los materiales de investigación junto a los borradores.
- **Pipeline de Publicación Automatizado**: Convierte automáticamente archivos Markdown al formato ICML, listos para importar en Adobe InDesign con estilos de párrafo y carácter académicos profesionales.
- **Sistema de Compilación para Desarrolladores**: Un sistema de compilación simple y confiable basado en Makefiles que proporciona validación de contenido, manejo de errores claro y procesamiento por lotes para mayor eficiencia.

## Arquitectura Técnica

El sistema se basa en un conjunto de herramientas de línea de comandos potentes y estándar, garantizando fiabilidad y portabilidad.

- **Pandoc**: Para una conversión de documentos robusta de Markdown a ICML.
- **Make**: Para automatizar todo el pipeline de compilación.
- **Bash y Perl**: Para scripting, procesamiento de texto y gestión de archivos.

Esta arquitectura crea un puente poderoso entre la escritura simple basada en texto y la maquetación profesional.
