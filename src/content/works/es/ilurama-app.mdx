# Ilurama: Herramientas para el Diseñador de Iluminación

Ilurama es una sofisticada aplicación móvil diseñada para diseñadores de iluminación, gaffers, directores de fotografía y fotógrafos. Sirve como un muestrario digital indispensable y una potente herramienta para crear y gestionar paletas de color para iluminación.

## Resumen del Proyecto

La aplicación proporciona una completa biblioteca digital de filtros de iluminación de varios fabricantes, permitiendo a los profesionales encontrar, comparar y organizar colores para sus proyectos. Más allá de una simple biblioteca, Ilurama introduce funciones innovadoras para la iluminación LED moderna, permitiendo a los usuarios definir colores personalizados mezclando parámetros CMY + UV y organizarlos en colecciones que se pueden compartir.

## Características Clave

- **Biblioteca Completa de Filtros:** Accede a una extensa base de datos de gelatinas y filtros de iluminación, con especificaciones técnicas detalladas, incluyendo gráficos de transmisión y datos colorimétricos (Y, x, y, abs).
- **Comparación Avanzada de Filtros:** Compara dos filtros lado a lado para analizar sus propiedades y ver cómo interactúan.
- **Creación de Color LED Personalizado:** Define colores LED precisos ajustando los parámetros de Cian, Magenta, Amarillo y UV, dándote control total sobre tu paleta de iluminación.
- **Sugerencias de Color Inteligentes:** Descubre colores análogos y complementarios para construir esquemas de iluminación sofisticados y armoniosos.
- **Colecciones y Compartir:** Organiza tus filtros favoritos y colores LED personalizados en colecciones. Compártelas con tu equipo y colaboradores para asegurar que todos estén en sintonía.
- **Interfaz de Usuario Intuitiva y Multiplataforma:** Desarrollada con Flutter, Ilurama ofrece una experiencia fluida y responsiva en dispositivos iOS y Android.

## Stack Técnico

Desarrollada con Flutter y Dart, la aplicación aprovecha prácticas modernas de desarrollo móvil para entregar una experiencia fluida y eficiente en todas las plataformas.
