# Hey Siri, do you love me?

This project showcases a contemporary dance choreography on the use of Siri as a source of personal well-being.

## Project Overview

The choreography was built to showcase the use of Siri as a source of personal well-being. The main requirements were:

- **Contemporary dance** with smooth animations
- **Siri** as a source of personal well-being
- **Personal** well-being
- **Performance** optimized for fast loading times

## Technical Stack

### Frontend Framework

The project is built with **Next.js 15** using the App Router architecture. This provides:

- Server-side rendering for optimal SEO
- Static site generation for fast page loads
- Modern React 19 features
- Built-in image optimization
